# TripMate

TripMate is a web application for planning trips with friends. You can create trips, add friends, manage dates, hotels, and keep track of your plans and all in one place.

## Features
- User authentication (login/register)
- Create, edit, and delete trips
- Add friends to trips
- Responsive design with nice UI
- Loading indicators and notifications

## How to Run Locally
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/tripmate.git`
2. Install dependencies: `npm install`
3. Start the frontend: `npm start`
4. Start the backend: follow your backend setup
